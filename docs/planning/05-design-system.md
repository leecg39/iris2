# Design System: IRIS 자동 매칭 앱

## 1. 디자인 원칙

- **신뢰감**: 정부지원사업 매칭이라는 특성상 전문적이고 신뢰가 가는 느낌
- **명확함**: 적합도 점수, 마감일 등 핵심 정보가 한눈에 보여야 함
- **효율성**: 최소 탭으로 원하는 정보에 도달

## 2. 컬러 시스템

| 용도 | 색상 | HEX |
|------|------|-----|
| Primary | 파란색 (신뢰, 정부) | #1565C0 |
| Secondary | 네이비 | #0D47A1 |
| Accent | 오렌지 (CTA, 적합도 강조) | #FF6D00 |
| Success | 초록 (높은 적합도) | #2E7D32 |
| Warning | 노랑 (마감 임박) | #F9A825 |
| Error | 빨강 (마감/에러) | #C62828 |
| Background | 밝은 회색 | #F5F5F5 |
| Surface | 흰색 | #FFFFFF |
| Text Primary | 다크그레이 | #212121 |
| Text Secondary | 미디엄그레이 | #757575 |

## 3. 타이포그래피

| 용도 | 크기 | 굵기 |
|------|------|------|
| 제목 (H1) | 24sp | Bold |
| 부제목 (H2) | 20sp | SemiBold |
| 본문 | 16sp | Regular |
| 캡션 | 14sp | Regular |
| 라벨 | 12sp | Medium |

- 한글 폰트: Pretendard 또는 Noto Sans KR

## 4. 컴포넌트

### 매칭 카드
- 공고명 (H2, Bold)
- 적합도 게이지 바 (색상으로 구분: 80%+ 초록, 50-79% 오렌지, 50% 미만 회색)
- 마감 D-day 뱃지 (D-3 이하 빨강)
- 공고기관명 (캡션)

### 적합도 점수 표시
- 원형 프로그래스 바 (큰 숫자 중앙)
- 색상 그라데이션 (낮음 → 높음)

### 하단 탭바
- 4탭: 홈 / 매칭 / 보고서 / 설정
- 아이콘 + 텍스트 라벨

## 5. 간격 시스템

- 기본 단위: 4px
- 카드 패딩: 16px
- 섹션 간격: 24px
- 화면 좌우 패딩: 16px
