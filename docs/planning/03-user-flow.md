# User Flow: IRIS 자동 매칭 앱

## 1. 전체 흐름

```
[앱 설치]
    ↓
[온보딩 (3장)]
    ↓
[사업자번호 입력]
    ↓
[기업 프로필 설정]
  ├── 공공API 자동 조회 (회사명, 업종, 매출 등)
  └── 사용자 추가 입력 (연구분야, 기술 키워드)
    ↓
[홈 대시보드]
  ├── 적합 공고 수
  ├── 마감 임박 공고
  └── 최신 매칭 결과
    ↓
[매칭 탭] ← AI가 IRIS 공고 수집 + 적합도 분석
  └── 매칭 목록 (공고명 + 적합도% + D-day)
        ↓
      [공고 상세]
        ├── 공고 전문
        ├── AI 요약
        ├── 적합도 분석 결과
        ├── 첨부파일 다운로드
        ├── [보고서 PDF 다운로드]
        └── [전문가 연결 버튼]
              ↓
            [전문가 상담 폼]
              ├── 자동채움 (프로필 + 공고 정보)
              ├── 메시지 작성
              └── [제출] → Notion DB 저장 + 확인 이메일 발송
```

## 2. 화면별 시나리오

### 시나리오 1: 첫 사용 (온보딩 → 매칭)
1. 앱 설치 후 첫 실행
2. 온보딩 3장 (앱 소개, 기능 안내, 시작하기)
3. 사업자번호 입력
4. 기업 정보 자동 조회 → 프로필 편집 화면
5. 연구분야/기술 키워드 추가 입력
6. 홈 대시보드 진입
7. 매칭 탭으로 이동 → AI 분석 시작
8. 결과 목록 확인 → 공고 상세 → AI 요약 확인

### 시나리오 2: 전문가 상담 신청
1. 매칭 목록에서 관심 공고 선택
2. 공고 상세 화면에서 적합도 확인
3. "전문가 연결" 버튼 탭
4. 자동채움 폼 확인 + 메시지 작성
5. 제출 → Notion DB 저장
6. 확인 이메일 수신

### 시나리오 3: 보고서 다운로드
1. 공고 상세에서 "보고서 다운로드" 탭
2. PDF 생성 (기업정보 + 공고정보 + 적합도 분석)
3. 기기에 저장
4. 보고서 탭에서 이력 확인

## 3. 화면 간 이동 맵

```
온보딩 → 사업자번호 입력 → 프로필 설정 → 홈
                                              ↕
홈 ←→ 매칭 ←→ 보고서 ←→ 설정     (하단 탭바)
       ↓
    공고 상세 → 전문가 상담 폼
       ↓
    보고서 PDF 다운로드

설정 → 프로필 편집 (프로필 설정 화면 재사용)
설정 → 알림 설정
```
