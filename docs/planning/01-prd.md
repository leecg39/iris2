# PRD: IRIS 정부지원사업 자동 매칭 앱

## MVP 캡슐

1. **목표**: 사업자번호 입력만으로 기업에 적합한 정부지원사업을 AI가 자동 매칭해주는 Flutter 앱
2. **페르소나**: 정부지원사업에 지원하려는 중소기업 대표, 스타트업 대표, R&D 담당자
3. **핵심 기능**: FEAT-1(기업 프로필 연동), FEAT-2(IRIS 공고 수집 + AI 매칭), FEAT-3(보고서 다운로드 + 전문가 연결)
4. **성공 지표 (노스스타)**: 주간 매칭 분석 완료 횟수 (사용자당)
5. **입력 지표**: 기업 프로필 등록 수, 전문가 상담 신청 수
6. **비기능 요구**: IRIS 스크래핑 응답시간 10초 이내, LLM 분석 30초 이내
7. **Out-of-scope**: 사업계획서 자동 작성, 결제 시스템, 전문가 매칭 알고리즘
8. **Top 리스크**: IRIS 사이트 구조 변경 시 스크래핑 실패
9. **완화/실험**: robots.txt 허용 경로 확인 + 스크래핑 헬스체크 모니터링
10. **다음 단계**: /screen-spec으로 화면별 상세 명세 생성

---

## 1. 프로젝트 개요

### 1.1 배경
매년 수십조 원의 정부 R&D 과제 및 지원사업이 공고되지만, 기업 입장에서 자신에게 맞는 공고를 찾는 것은 매우 비효율적인 수작업입니다. IRIS(https://www.iris.go.kr/main.do) 사이트에 매일 접속하여 키워드 검색, 마감 확인, 첨부파일 열람을 반복해야 합니다.

### 1.2 솔루션
사업자번호 입력 → 공공API로 기업 데이터 자동 연동 → IRIS 공고 데이터 수집 → LLM이 기업과 공고를 분석하여 적합도 점수 산출 → 매칭 결과 + 보고서 + 전문가 연결까지 원스톱으로 제공하는 Flutter 앱.

### 1.3 타겟 사용자
- 중소기업 대표/임원
- 스타트업 대표
- 기업 내 R&D/사업기획 담당자

### 1.4 수익 모델
- MVP는 무료로 시작
- 추후 프리미엄 기능(전문가 연결 수수료, 고급 분석) 검토

## 2. 핵심 기능

### FEAT-1: 기업 프로필 연동
- **설명**: 사업자번호 입력 → 공공API(기업마당 등)로 기업 기본정보(회사명, 업종, 매출 등) 자동 조회
- **사용자 추가 입력**: 연구분야, 기술 키워드 (매칭 정확도 향상용)
- **우선순위**: P0 (필수)

### FEAT-2: IRIS 공고 수집 + AI 적합도 분석
- **설명**: IRIS(https://www.iris.go.kr/main.do)에서 현재 진행 중인 공고 데이터 스크래핑
- **AI 분석**: 기업 데이터 + 공고 요건을 LLM(OpenAI GPT)에 전달하여 적합도 점수(%) 산출
- **필터링**: 진행 중/마감 전, 기간별 필터
- **우선순위**: P0 (필수)

### FEAT-3: 매칭 결과 + 보고서 + 전문가 연결
- **매칭 결과**: 공고명 + 적합도 점수 + 마감일(D-day) 목록
- **공고 상세**: 공고 전문 + AI 요약 + 적합도 분석 결과 + 첨부파일 다운로드
- **보고서**: 매칭 분석 결과를 PDF로 다운로드
- **전문가 연결**: 사업계획서 작성 전문가에게 상담 신청 (자동채움 폼 → DB 저장 → 확인 이메일 발송)
- **우선순위**: P0 (필수)

## 3. 유사 서비스 분석 (벤치마킹)

| 서비스 | 특징 | 차별화 포인트 |
|--------|------|---------------|
| IRIS 직접 검색 | 키워드 수동 검색, 필터 제한적 | 우리: AI가 기업 데이터 기반 자동 매칭 |
| 바이브랩스 NTIS 앱 (참고 영상) | 키워드 기반 스크래핑 + AI 요약 | 우리: 사업자번호 기반 자동 프로필 + 적합도 점수 |
| 기업마당 | 정부지원사업 검색 포털 | 우리: 개인화된 매칭 + 보고서 + 전문가 연결 |

## 4. 가설 vs 사실

| 항목 | 유형 | 검증 방법 |
|------|------|----------|
| 공공API로 기업 기본정보 조회 가능 | 가설 | 기업마당 API 테스트 |
| IRIS robots.txt에서 스크래핑 허용 경로 존재 | 가설 | robots.txt 직접 확인 |
| LLM이 공고-기업 매칭을 합리적으로 수행 가능 | 가설 | GPT-4o에 샘플 데이터 테스트 |
| 사용자가 적합도 점수를 신뢰함 | 가설 | MVP 사용자 피드백 |
| Flutter로 iOS/Android 동시 지원 가능 | 사실 | Flutter 공식 지원 |

## 5. 성공 지표

### 노스스타 메트릭
- **주간 매칭 분석 완료 횟수** (사용자당)

### 입력 지표
- 기업 프로필 등록 완료율
- 전문가 상담 신청 전환율

### 가드레일
- 스크래핑 실패율 < 5%
- LLM 분석 정확도 (사용자 피드백 기반)
